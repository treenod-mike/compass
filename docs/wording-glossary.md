# Wording Glossary — 전문용어 → 평이어 전환 사전

**목적**: Compass 전체 UI에서 비전문가도 이해할 수 있는 한국어 카피로 통일.
**원칙**:
1. **명사형 약어**(ROAS, MOIC, DPI)는 **유지하되 첫 노출 시 풀어 설명**
2. **통계 용어**(credible interval, P10/P50/P90)는 **평이어로 번역**
3. **영어 명령형**(Invest, Hold, Reduce)은 **한글 동사**로
4. **달러 금액**은 **원화 환산** 또는 **단순 배수**로 병기
5. **톤**: 친근 + 전문 중간 — "~해요" 체 기본, 숫자는 tabular

---

## 1. 금융 / 투자 용어 (Finance)

| 현재 | 평이어 | 유지 조건 |
|------|-------|----------|
| MOIC (Multiple on Invested Capital) | 투자 대비 자산 배수 / "1원이 N원으로 불었어요" | 차트 축 라벨·리포트 |
| ROAS (Return on Ad Spend) | 광고비 회수율 / "100원 써서 N원 벌어요" | 요약 strip 라벨 |
| Blended ROAS | 전체 광고 회수율 | 동일 |
| LTV (Lifetime Value) | 1인당 생애 매출 | 개발 리포트 |
| CPI (Cost per Install) | 설치당 광고비 | 마케팅 리포트 |
| DPI (Distributions to Paid-In) | 투자 회수 비율 / "돌려받은 돈 배수" | VC 리포트 |
| IRR (Internal Rate of Return) | 연환산 수익률 | 재무 리포트 |
| NPV (Net Present Value) | 현재 가치 기준 순이익 | 재무 |
| ARR (Annual Recurring Revenue) | 연 매출 전망 | SaaS 컨텍스트만 |
| Runway | 자금 소진까지 남은 기간 / "현금 X개월 더 버텨요" | 차트 라벨 |
| Burn Rate | 월 현금 소진 / "매달 쓰는 돈" | 재무 |
| Break-even Point (BEP) | 본전 시점 / "손해·이익 갈림점" | 차트 라벨 |
| Payback Window | 본전 회수 기간 / "투자금 돌려받는 시간" | 차트 라벨 |

## 2. 통계 / 베이지안 용어 (Statistics)

| 현재 | 평이어 | 유지 조건 |
|------|-------|----------|
| P10 / P50 / P90 | 빠르면 / 평균 / 늦으면 | 차트 축·툴팁 |
| Credible Interval | 가능성 범위 / "95% 확률로 이 사이" | 차트 보조 설명 |
| Confidence Level 78% | "10번 중 8번은 맞아요" / "확신 정도 78점" | 수치 병기 |
| Prior | 시장 기대치 / 업계 평균 | 차트 라벨 |
| Posterior | 우리 데이터 반영한 예측 | 차트 라벨 |
| Bayesian | 경험 + 데이터로 보정하는 | 기술 설명만 |
| Uncertainty Band | 불확실성 범위 / "위아래 여유폭" | 차트 |
| Monte Carlo | 수천 번 시뮬레이션 결과 | 차트 메타 |
| Median | 중간값 | 차트 |
| P-value | 우연일 확률 | 실험 |

## 11. 적용 가이드

### 11.1 "첫 노출 시 풀어 쓰기" 규칙
```
❌ "MOIC 1.27x"
✅ "투자 배수 1.27x (투자한 돈이 1.27배 됨)"
```

### 11.2 "수치 + 평이어 병기" 규칙
```
❌ "신뢰도 78%"
✅ "신뢰도 78% · 10번 중 8번 맞아요"
```

### 11.3 "영문 상태 뱃지 단독 사용 금지"
```
❌ [INVEST]
✅ [🚀 확대] 또는 [확대 · INVEST]
```

### 11.4 "통계 용어는 시각화에만"
```
❌ "Payback P50: 44d [P10: 35d – P90: 58d]"
✅ "평균 44일이면 본전 · 빠르면 35일, 늦어도 58일"
```
